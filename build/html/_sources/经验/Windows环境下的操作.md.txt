# Windows环境下的操作

### Windows操作系统快捷键

ctrl + c 复制

ctrl + v 粘贴

ctrl + z 回退

win + r 打开运行命令窗口，输入cmd，打开命令行窗口

win + d 桌面最小化

win + e 打开我的电脑

alt + tab 不同窗口之间的切换，按的顺序是按住alt，请点tab。练习两个窗口之间的来回切换，提高学习效率。

win + <---  或者  win + --->  分屏显示，在一个屏幕下，看多个窗口的常用操作



### Windows下使用Dev C++

关闭符号自动补全

关闭代码自动补全

不要改乱七八糟的字体，尽量使用自带

代码字体大小，使用Ctrl + 滚轮



### Windows电脑去除电脑广告

很多学生用浏览器用的是360浏览器，而这个浏览器的广告实在太多，蹦出来频次高，内容娱乐化强，影响注意力集中。一般情况下，学习期间，尽量使用Chrome浏览器。

虽然360浏览器是一个好的浏览器，作为360公司的三级火箭中很重要的一部分，但这并不适合学生使用。

电脑上的杀毒软件不用安那么多，个人电脑没必要搞那么多，就安一个360防火墙足够。

还有很多学生，安装一些游戏，一顿安装，被自动安装了一个小软件，这些软件定期使用360软件卸载，清理清理。
